# Ministros OS Simulator

**Ministros OS Simulator** is a text-based operating system shell written in C. It features a login screen, command-line interface, and simulated system commands to give a fun and educational experience of using an OS shell.

## Features

- Login screen with username and password (you can enter **any username and password**)  
- Command-line interface with simulated commands:
  - `whoami` – prints the username  
  - `--version` – prints Ministros OS version  
  - `help` – shows available commands  
  - `sudo pacman -Syu` – simulates system update  
  - `shutdown now` / `exit` – exit the shell  
- Boot splash screen from `splash.txt`  
- Lightweight and easy to modify
